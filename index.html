Simple Graph
  • แต่ละเส้นเชื่อมต้องเชื่อมสองจุดยอดที่ต่างกัน และไม่มีสองเส้นเชื่อมจุดคู่เดียวกันซ้ำ
  • simple graph จะไม่มีทิศทาง, ไม่มีเส้นขนาน, ไม่มีการ loop เข้าตัวเอง
Simple Directed Graph
  • เส้นเชื่อมทุกเส้นจะมีลูกศรบอกทิศทาง
  • ถ้าเส้นเชื่อมระบุเป็นคู่อันดับ (u, v) จะหมายถึง เริ่มวิ่งจากจุด u ไปสิ้นสุดที่จุด v จะย้อนศรไม่ได้
  • กราฟที่จุดปลายของเส้นเชื่อมไม่มีลำดับ เรียกว่า undirected graph
Multigraph
  • มีเส้นเชื่อมขนาน (parallel edges) ระหว่างจุดยอดคู่เดียวกันได้มากกว่า 1 เส้น
  • ถ้ามี m เส้นเชื่อมระหว่าง u, v เรียกว่า {u, v} มี multiplicity m
  • ถ้ามี loop ปนอยู่ด้วย จะเรียกว่า pseudograph
  • เช่น จำนวนเที่ยวบินที่ให้บริการระหว่างสองเมืองในหนึ่งวัน
Pseudograph
  • Pseudograph คือ undirected graph ที่อนุญาตให้มีทั้ง เส้นขนาน (parallel edges) และ loop
  • Loop คือเส้นที่เชื่อมจุดยอดกับตัวมันเอง — loop หนึ่งเส้น นับ degree เพิ่ม 2
  • Pseudograph ≠ Multigraph — multigraph มีเส้นขนานได้แต่ไม่มี loop
  • ใช้จำลอง network ที่มี diagnostic link วนกลับเข้า data center ตัวเอง
Directed Multigraph
  • Directed Multigraph คือกราฟที่มีทิศทาง และ อนุญาตให้มีเส้นขนาน (parallel directed edges) รวมถึง loop ได้
  • เส้นขนานที่มีทิศทางเดียวกันระหว่าง u→v สองเส้น หมายถึงมี 2 ช่องทางที่วิ่งในทิศทางเดียวกัน
  • ต่างจาก Simple Directed — ที่นี่ระหว่าง u, v มีได้หลาย arc ในทิศทางเดิม
  • เป็นโมเดลทั่วไปที่สุดในกลุ่ม directed graph ก่อนจะถึง Mixed Graph
Mixed Graph
  • Mixed Graph G = (V, E, A) ประกอบด้วยทั้ง undirected edges (E) และ directed edges (A) ในกราฟเดียวกัน
  • เส้นบางเส้นมีทิศทาง บางเส้นไม่มีทิศทาง เลือกได้ตามความเหมาะสมของ relation

ตัวอย่างของ Graph model ทุกประเภทที่อิงจาก Computer Network

เปรียบเทียบโมเดล Computer Network ของแต่ละประเภทกราฟ

Simple Graph

เครือข่ายสนใจแค่ว่า data center มีหรือไม่มี link เชื่อม — ไม่สนจำนวน

ไม่มี direction ไม่มี parallel ไม่มี loop
Simple Directed Graph

สนใจทิศทางการไหลของข้อมูล — SF ส่งไป Denver แต่ Denver ไม่ได้ส่งกลับ

มี direction ไม่มี parallel ไม่มี loop
Multigraph

สนใจจำนวน link — Chicago–NY มี 3 ช่องทาง, SF–Denver มี 2 ช่องทาง

ไม่มี direction มี parallel ไม่มี loop
Pseudograph

data center มี diagnostic loop วนตัวเอง + มีสายสำรองระหว่าง SF–Chicago 2 เส้น

ไม่มี direction มี parallel มี loop
Directed Multigraph

SF มี self-diagnostic loop + SF→Chicago มี 2 directed link + Chicago↔Wash สวนทางกัน

มี direction มี parallel มี loop
Mixed Graph

SF↔Chicago เป็น fiber สองทาง, NY→Houston เป็น satellite uplink ทางเดียว

direction บางเส้น มี parallel มี loop
