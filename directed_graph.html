Directed Graph Terminology

Adjacent · In-degree · Out-degree · Theorem 3 — บทที่ 3

Definition 4 Directed Graph

เมื่อ (u, v) เป็น edge ใน Directed Graph G, เราบอกว่า u เป็น adjacent ถึง v และ v เป็น adjacent จาก u

  • u เรียกว่า Initial vertex ของ (u, v)
  • v เรียกว่า Terminal vertex หรือ End vertex ของ (u, v)
  • ถ้าเป็น Loop (วน edge กลับตัวเอง) — Initial vertex และ Terminal vertex จะเป็นจุดเดียวกัน
ลูกศรชี้จาก Initial ไปยัง Terminal
Directed Edge (u → v)
Definition 5 In/Out-degree

deg⁻(v) — In-degree

จำนวน edge ที่มี v เป็น Terminal vertex
นับเส้นที่ "เข้า" มาหา v

deg⁺(v) — Out-degree

จำนวน edge ที่มี v เป็น Initial vertex
นับเส้นที่ "ออก" จาก v

Loop ที่จุดยอด v เพิ่ม deg⁻(v) และ deg⁺(v) อย่างละ 1
กดจุดยอดเพื่อดู degree
กดที่จุดยอดบนกราฟ...

ตัวอย่าง — หา In-degree และ Out-degree

Graph G มีจุดยอด a, b, c, d, e, f

Worked Example
Graph G
Vertex deg⁻ (in) deg⁺ (out)
a 2 2
b 2 2
c 3 3

รวม 12 12
สังเกต: Σ deg⁻ = Σ deg⁺ = 12 = จำนวน edges ทั้งหมด
(นับ loop ที่ c เป็น 1 in + 1 out)

Theorem 3 — Sum of In/Out-degrees

ผลรวม in-degree = ผลรวม out-degree = จำนวน edges

Theorem 3

ให้ G = (V, E) เป็น Graph ที่มี directed edges แล้ว:

|E|  =  Σ deg⁻(v)  =  Σ deg⁺(v)

กล่าวคือ ผลรวมของ in-degree ทุกจุดยอด เท่ากับผลรวม out-degree ทุกจุดยอด และเท่ากับจำนวน edge ทั้งหมดเสมอ

Proof (ไอเดีย):
เส้นเชื่อมแต่ละเส้นมี initial vertex 1 จุด (เพิ่ม out-degree 1 ครั้ง) และ terminal vertex 1 จุด (เพิ่ม in-degree 1 ครั้ง) ดังนั้น ผลรวมทั้งสองเท่ากับจำนวน edges เสมอ ∎

ลองเพิ่ม edge แล้วดูผลรวม

deg⁻(P)=0 deg⁺(P)=0
deg⁻(Q)=0 deg⁺(Q)=0
deg⁻(R)=0 deg⁺(R)=0
deg⁻(S)=0 deg⁺(S)=0
|E| = 0 Σdeg⁻ = 0 Σdeg⁺ = 0

Special Types of Simple Graphs

Complete Graph · Cycle · Wheel · Hypercube

Complete Graph Kₙ

Complete Graph Kₙ คือ Simple Graph บน n จุดยอด ที่มีเส้นเชื่อมระหว่างทุกคู่ของจุดยอดที่ต่างกัน ครบทุกคู่

  • จำนวน edge = n(n−1)/2
  • ทุกจุดมี degree = n−1
  • Graph ที่ไม่ complete เรียกว่า Noncomplete
K₂: 1 edge  |  K₃: 3 edges  |  K₄: 6 edges  |  K₅: 10 edges
Cycle Cₙ

Cycle Cₙ (n ≥ 3) ประกอบด้วย n จุดยอด v₁, v₂, …, vₙ และมี edges {v₁,v₂}, {v₂,v₃}, …, {vₙ₋₁,vₙ}, {vₙ,v₁}

  • จำนวน edge = n
  • ทุกจุดมี degree = 2
  • รูปร่างเป็น "วงกลม" เชื่อมต่อกันครบ
Wheel Wₙ

Wheel Wₙ ได้จากการเพิ่มจุดยอด hub 1 จุดเข้าไปใน Cycle Cₙ (n ≥ 3) แล้วต่อ edge จาก hub ไปยังทุกจุดใน Cₙ

  • จำนวนจุดยอดทั้งหมด = n+1
  • จุด hub มี degree = n
  • จุดรอบนอกมี degree = 3
n-dimensional Hypercube Qₙ

Hypercube Qₙ คือ Graph ที่มีจุดยอดแทน bit string ยาว n ทั้งหมด 2ⁿ จุด

สองจุดยอดจะ adjacent กัน ก็ต่อเมื่อ bit string ที่แทนต่างกัน เพียง 1 bit เท่านั้น

  • Q₁: 2 จุด, 1 edge (0─1)
  • Q₂: 4 จุด, 4 edges (00,01,10,11)
  • Q₃: 8 จุด, 12 edges (000–111)
